### Week 4 objectives:

* Learn Amazon S3 and the concepts of bucket, object, storage class, and versioning.
* Practice data access control using bucket policies and IAM policies.
* Deploy a simple static website using Amazon S3.

### Week 4 outcomes:

* Understood that Amazon S3 is an object storage service for static data, backups, and media assets.
* Created and managed buckets, objects, logical folders, and basic versioning.
* Learned the risks of public buckets and the importance of Block Public Access.
* Deployed a simple static website using S3.
* Identified the role of S3 in the movie website project: storing posters, cover images, videos, and frontend assets.
