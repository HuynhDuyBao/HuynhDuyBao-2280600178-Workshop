### Week 5 objectives:

* Learn Amazon RDS and how AWS supports managed relational databases.
* Practice creating a database, configuring security, and connecting from an application.
* Prepare the initial data model for the movie website.

### Week 5 outcomes:

* Understood that RDS reduces database operation tasks such as backup, patching, and basic configuration.
* Created and connected to a test database on AWS.
* Learned how to restrict database access using Security Groups.
* Built the initial data model for the movie website.
* Gained experience controlling RDS cost by selecting small configurations and deleting resources after lab practice.
